### git是什么

git是版本控制工具，类似的软件有svn、cvs、vss，git比他们强大很多！

### 使用代理

在使用git前输入如下命令即可：

```
export http_proxy="http://用户名:密码@服务器地址:端口"
export https_proxy="http://用户名:密码@服务器地址:端口"
```
